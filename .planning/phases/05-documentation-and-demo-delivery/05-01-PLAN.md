<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Source API schemas -> public docs | Current FastAPI routes and Pydantic schemas become integration contracts for Java callers. |
| CLI import workflow -> reviewer docs | Operator-only commands can be mistaken for backend HTTP integration if not clearly labeled. |
| Search score semantics -> Java/demo consumers | Ranking fields can be misread as medical confidence if docs use unsafe language. |
| Documentation examples -> user copy/paste | Java/curl examples will be copied directly into integration work. |

## STRIDE/Safety Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01-01 | Spoofing | `docs/API.md` route docs | mitigate | Add OpenAPI contract test to verify docs only claim real HTTP routes and explicitly say import/rebuild are CLI-only. |
| T-05-01-02 | Tampering | Java/curl examples | mitigate | Test examples contain current request/response field names and real endpoint paths. |
| T-05-01-03 | Safety/Misuse | score semantics docs | mitigate | Require exact score semantics sentence and v1 no diagnosis/prescribing language in README/API docs. |
| T-05-01-04 | Repudiation | error docs | mitigate | Document stable `SearchErrorEnvelope` and known `validation_error`, `index_not_ready`, provider, vector, and reranker codes. |
</threat_model>
